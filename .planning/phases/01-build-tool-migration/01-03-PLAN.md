**B. Replace all 14 Vue.set() calls with direct assignment:**

Vue 3's Proxy-based reactivity tracks direct property assignment on reactive objects. Every `Vue.set(obj, key, val)` becomes `obj[key] = val`.

Specific replacements in mutations:

1. `updateAbilityScore`: `Vue.set(state.abilities[i], 'score', payload.score)` -> `state.abilities[i].score = payload.score`

2. `updateSkillProficiency` (two calls):
   - `Vue.set(state.skills[payload.i], 'proficient', payload.proficient)` -> `state.skills[payload.i].proficient = payload.proficient`
   - `Vue.set(state.skills[payload.i], 'doubleProficient', payload.doubleProficient)` -> `state.skills[payload.i].doubleProficient = payload.doubleProficient`

3. `updateSavingThrow`: `Vue.set(state.savingThrows[i], 'proficient', payload.proficient)` -> `state.savingThrows[i].proficient = payload.proficient`

4. `updateCoins`: `Vue.set(state.coins[payload.i], 'amount', payload.amount)` -> `state.coins[payload.i].amount = payload.amount`

5. `updateListField` (two calls):
   - `Vue.set(state[payload.field][payload.i], 'val', payload.val)` -> `state[payload.field][payload.i].val = payload.val`
   - `Vue.set(state[payload.field][payload.i], 'collapsed', payload.collapsed)` -> `state[payload.field][payload.i].collapsed = payload.collapsed`

6. `updateSpellName`: `Vue.set(state[payload.field].spells[payload.i], 'name', payload.name)` -> `state[payload.field].spells[payload.i].name = payload.name`

7. `updateSpellPrepared`: `Vue.set(state[payload.field].spells[payload.i], 'prepared', payload.prepared)` -> `state[payload.field].spells[payload.i].prepared = payload.prepared`

8. `updateSpellCollapsed`: `Vue.set(state[payload.field].spells[payload.i], 'collapsed', payload.collapsed)` -> `state[payload.field].spells[payload.i].collapsed = payload.collapsed`

9. `updateSpellSlots`: `Vue.set(state[payload.field], 'slots', payload.val)` -> `state[payload.field].slots = payload.val`

10. `updateExpendedSlots`: `Vue.set(state[payload.field], 'expended', payload.val)` -> `state[payload.field].expended = payload.val`

**B. Rewrite `js/app.js`** (per VUE-01, VUE-03, VUE-05, VUE-10):

Replace the entire file with:

```javascript
import { createApp } from 'vue';
import mitt from 'mitt';
import Sheet from './components/Sheet.vue';
import store from './store';
import { signedNumString } from './utils';
import { i18nPlugin } from './i18n';

/* -- Event bus (replaces new Vue() instance) -- */
window.sheetEvent = mitt();

window.md = window.markdownit({
  html: true,
  linkify: true,
  typographer: true,
});

const app = createApp(Sheet);
app.use(store);
app.use(i18nPlugin);

// Register signedNumString as a global property (replaces Vue.filter)
// Components access via this.$signedNumString() or template: $signedNumString()
app.config.globalProperties.$signedNumString = signedNumString;

app.mount('#sheet');
```

Key changes:
- `import { createApp } from 'vue'` replaces `import Vue from 'vue'` (VUE-01)
- `mitt()` replaces `new Vue()` for event bus (VUE-03) -- mitt uses `.emit()/.on()/.off()` without `$` prefix
- `createApp(Sheet)` + `app.mount('#sheet')` replaces `new Vue({ el: '#sheet', render: h => h(Sheet) })` (VUE-10)
- `app.use(store)` replaces `Vue.use(Vuex)` in store.js (Vuex registered at app level)
- `app.use(i18nPlugin)` replaces `Vue.use(i18nPlugin)` (plugin registered at app level)
- `Vue.filter('signedNumString', signedNumString)` removed -- Vue 3 has no filters (VUE-05). Instead, `signedNumString` is added as `app.config.globalProperties.$signedNumString` so components can call `$signedNumString(value)` in templates
- Component import uses `.vue` extension: `'./components/Sheet.vue'` (Vite requires explicit extensions for SFC imports)

**D. Update `js/i18n.js`** (per VUE-04):

Replace the `install` method in the `i18nPlugin` export:

```javascript
// BEFORE:
export const i18nPlugin = {
  install(Vue) {
    Vue.prototype.$t = t;
    Vue.prototype.$getLocale = getLocale;
    Vue.prototype.$setLocale = setLocale;
  }
};

// AFTER:
export const i18nPlugin = {
  install(app) {
    app.config.globalProperties.$t = t;
    app.config.globalProperties.$getLocale = getLocale;
    app.config.globalProperties.$setLocale = setLocale;
  }
};
```

Vue 3 plugins receive the app instance, not the Vue constructor. `Vue.prototype` becomes `app.config.globalProperties`.
