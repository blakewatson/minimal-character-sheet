Create a new file `js/export.js` with the following exported functions:

1. **`sanitizeFilename(name)`** — per D-04:
   - If `name` is falsy or only whitespace, return `'character'`
   - Trim the name
   - Remove illegal filename characters: `< > : " / \ | ? *` and control chars (regex: `/[<>:"/\\|?*\x00-\x1f]/g`)
   - Replace whitespace runs with hyphens (regex: `/\s+/g` -> `'-'`)
   - Truncate to 100 characters
   - If result is empty after sanitization, return `'character'`

2. **`downloadFile(content, filename, mimeType)`** — client-side Blob download:
   ```javascript
   function downloadFile(content, filename, mimeType) {
     const blob = new Blob([content], { type: mimeType });
     const url = URL.createObjectURL(blob);
     const a = document.createElement('a');
     a.href = url;
     a.download = filename;
     document.body.appendChild(a);
     a.click();
     document.body.removeChild(a);
     URL.revokeObjectURL(url);
   }
   ```

3. **`extractQuillText(delta)`** — extract plain text from Quill delta objects for Markdown export (per D-03, D-11):
   - If `delta` is falsy or is an empty object (no `ops` property), return `''`
   - If `delta` is a string, return it as-is (some fields may store plain strings)
   - If `delta.ops` is an array, iterate and concatenate each op's `insert` value where `typeof insert === 'string'`
   - Trim trailing newlines
   - Return the concatenated text

5. **`exportSheet(slug, name)`** — the main export function, per D-01, D-02, D-09, D-14:
   - Fetch sheet data: `const resp = await fetch('/sheet-data/' + slug); const json = await resp.json();`
   - If `!json.success || !json.sheet`, log error and return
   - Extract `sheetData = json.sheet.data` (this is the full JSON blob, same as autosave per D-09)
   - Extract `is_2024 = json.sheet.is_2024`
   - Build the sanitized filename base: `const fileBase = sanitizeFilename(name)`
   - Download JSON: `downloadFile(JSON.stringify(sheetData, null, 2), fileBase + '.json', 'application/json')` -- per D-02, uses JSON.stringify on the store data
   - Download Markdown: `downloadFile(generateMarkdown(sheetData, is_2024), fileBase + '.md', 'text/markdown')` -- per D-03
   - Use `setTimeout(() => { /* second download */ }, 100)` between the two downloads to avoid browser blocking

Export only `exportSheet` and `sanitizeFilename` (sanitizeFilename will be reused by import for success messages). Keep `downloadFile`, `extractQuillText`, `generateMarkdown` as non-exported module functions.
