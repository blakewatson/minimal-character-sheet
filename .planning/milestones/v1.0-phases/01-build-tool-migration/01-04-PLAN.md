**B. Wrap Quill editor with markRaw() (VUE-07) -- 1 file:**

In `js/components/QuillEditor.vue`:

1. Add import at top of `<script>`: `import { markRaw } from 'vue';`

2. In the `initQuill()` method, wrap the Quill constructor result:
```javascript
// BEFORE:
this.editor = new Quill(this.$el, { ... });

// AFTER:
this.editor = markRaw(new Quill(this.$el, { ... }));
```

This prevents Vue 3's proxy from wrapping the Quill instance, which would interfere with Quill's internal DOM manipulation.

**C. Replace template filter syntax with function calls (VUE-05) -- 6 files:**

Vue 3 removed the `|` filter syntax. The `signedNumString` function is registered as `app.config.globalProperties.$signedNumString` in the entry points (Plan 03). Replace every `{{ expression | signedNumString }}` with `{{ $signedNumString(expression) }}` in templates.

For `:value` bindings that use filters (Print.vue line 83), change from `:value="expression | signedNumString"` to `:value="$signedNumString(expression)"`.

Specific replacements:

1. **js/components/Ability.vue** line 11:
   - `{{ modifier | signedNumString }}` -> `{{ $signedNumString(modifier) }}`
   - line 40: `{{ saveBonus | signedNumString }}` -> `{{ $signedNumString(saveBonus) }}`
