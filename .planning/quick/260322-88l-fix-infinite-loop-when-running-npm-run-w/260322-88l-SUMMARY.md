# Quick Task 260322-88l: Fix infinite loop when running npm run watch

**One-liner:** Conditionally disable emptyOutDir during Vite watch mode using ROLLUP_WATCH env check

## What Was Done

Changed `emptyOutDir: true` to `emptyOutDir: !process.env.ROLLUP_WATCH` in `vite.config.js`. When Vite runs in watch mode, `ROLLUP_WATCH` is set to `"true"`, so `emptyOutDir` becomes `false`, preventing the watcher from detecting its own dist/ deletions and triggering infinite rebuilds. Normal builds (`npm run dev`, `npm run prod`) still clear the output directory.
