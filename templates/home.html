a minimal character sheet app for D&D 5e

Digital character sheets that are more convenient than PDFs, and more flexible than automated character sheets.

Features:

  • Calculates modifiers for abilities and skills
  • Unassuming — no limits on what features your character has.
  • Rich text fields that allow links and light formatting.
  • Important info is always a glance away.
  • Spell slot management.
  • Supports D&D 5e 2014 and 2024 rules.

Why should I use this?

There are already great options like D&D Beyond that support extremely detailed and automated character sheets. That said, you might want something simpler and less automated. Minimal character sheet aims to be more like paper—it's flexible and it doesn't limit what features and abilities you give your character. And because it handles rich text, you can still link to any external resources you like.

I made minimal character sheet because I wanted something more usable than a form fillable PDF, but more flexible than fully automated character sheets.
